\# AI-Based Posture Detection System



An intelligent posture detection system built using Python, OpenCV, and MediaPipe.



The system analyzes human posture in real time through a webcam, calculates body angles, and provides immediate feedback to help users maintain healthy sitting habits.



\---



\## Features



\- Real-time posture monitoring

\- Automatic user calibration

\- Shoulder angle analysis

\- Neck angle analysis

\- Hip angle analysis

\- Poor posture detection

\- Visual feedback and alerts

\- Statistics tracking

\- Logging system



\---



\## Technologies Used



\- Python

\- OpenCV

\- MediaPipe

\- NumPy

\- Pandas



\---



\## How It Works



1\. Capture video stream from webcam

2\. Detect body landmarks using MediaPipe Pose

3\. Calculate posture-related angles

4\. Compare angles against calibrated thresholds

5\. Detect poor posture conditions

6\. Display feedback and alerts in real time
